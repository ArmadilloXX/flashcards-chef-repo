##DESCRIPTION

This repo helps to set up development environment for Flashcards application or deploy it.

##Requirements
  - Vagrant Chef Plugin

##USAGE

  - Start Chef Zero
  - Upload your repo to Chef Zero with `knife upload .` from root repo directory
  - Run `vagrant up`

  ###OR

  - `knife serve --chef-zero-host IP`
  - Run `vagrant up`
